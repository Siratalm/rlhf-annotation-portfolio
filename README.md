## 🧠 What's Inside

### 1. Annotation Rubric (`guidelines/annotation_rubric.md`)
My personal scoring framework for evaluating LLM responses — the same criteria I apply across RLHF tasks. Covers:
- Helpfulness, accuracy, and instruction-following
- Safety and policy compliance
- Tone, fluency, and format quality

### 2. RLHF Preference Dataset (`data/rlhf_preference_dataset.csv`)
50 prompt + dual-response pairs with:
- Preference rankings (Response A vs B)
- Confidence scores
- Reasoning notes explaining *why* one response was preferred

### 3. Safety Feedback Samples (`data/safety_feedback_samples.csv`)
25 red-teaming examples showing:
- Harmful content detection
- Bias flagging
- Policy violation identification
- Recommended model behavior correction

### 4. Dataset Validator (`scripts/validate_dataset.py`)
A Python script that checks the dataset for:
- Missing fields
- Out-of-range scores
- Inconsistent labels
- Summary quality stats
